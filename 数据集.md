# M2DGR: A Multi-sensor and Multi-scenario SLAM Dataset for Ground Robots

​		M2DGR:一种新型的大规模数据集，由地面机器人收集，并配备了完整的传感器套件，包括6个鱼眼和一个指向天空的RGB相机、一个红外相机、一个事件相机、一个视觉-惯性传感器(VI-sensor)、一个惯性测量单元(IMU)、一个激光雷达(LiDAR)、一个消费级全球导航卫星系统(GNSS)接收机和一个具有实时运动学信号(RTK)的GNSS- imu导航系统。所有这些传感器都经过了良好的校准和同步，同时记录了它们的数据。地面真实轨迹由运动捕捉装置、激光三维跟踪器和RTK接收器获得。该数据集包含36个序列(约1TB)，在各种场景中捕获，包括室内和室外环境。我们在M2DGR上评估了最先进的SLAM算法。结果表明，现有的解决方案在某些场景下性能很差。

数据集地址：https://github.com/SJTU-ViSYS/M2DGR

贡献如下：
1、我们收集了地面机器人在室内和室外具有挑战性的序列，并配备了完整的传感器套件，其中包括6个环视鱼眼相机、一个指向天空的RGB相机、一个事件相机、一个红外相机、一个32线激光雷达、两个GNSS接收器和两个imu。据我们所知，这是第一个基于地面机器人导航的SLAM数据集。

2、我们在一些具有挑战性的场景中记录了轨迹，如电梯、完全黑暗，这很容易使现有的定位解决方案失败。这些情况在地面机器人应用中是常见的，但在以前的数据集中很少讨论。

3、我们提出了一个全面的地面机器人导航基准。在此基础上，我们评估了现有各种设计的最先进的SLAM算法，并分别分析了它们的特点和缺陷。





# LiDAR-Inertial 3D SLAM with Plane Constraint forMulti-story Building

​		一个多模态激光雷达数据集，其中包含不同扫描模式（旋转和固态）、传感技术和激光雷达相机的传感器。该数据集的重点是低漂移里程计，在室内和室外环境中都可以获得地面实况数据，运动捕捉 (MOCAP) 系统具有亚毫米精度。为了在更远的距离进行比较，还包括在室内和室外较大空间中记录的数据。数据集包含来自旋转激光雷达和固态激光雷达的点云数据。此外，它还提供来自高分辨率旋转激光雷达的距离图像、来自激光雷达相机的 RGB 和深度图像，以及来自内置 IMU 的惯性数据。

开源地址：https://utufi.sharepoint.com/sites/msteams_0ed7e9/Shared%20Documents/Forms/AllItems.aspx?id=%2Fsites%2Fmsteams%5F0ed7e9%2FShared%20Documents%2FDatasets%2FTIERS%5FMulti%5FLidar%5FDataset&p=true&ga=1

主要贡献：
1、一个数据集，其中包含来自 5 个不同激光雷达传感器和一个激光雷达相机的各种环境中的数据。据我们所知，就这些环境的激光雷达传感器而言，这是最多样化的数据集。该数据集包括具有 16 个（Velodyne VLP-16）、64 个（Ouster OS1-64）和 128 个（Ouster OS1-128）通道和不同垂直 FoV 的旋转激光雷达。还包括两个具有不同扫描模式和 FoV 的不同固态激光雷达（Livox Horizon 和 Livox Avia）。激光雷达摄像头 (RealSense L515) 提供 RGB 图像和激光雷达辅助深度图像。来自 Ouster 传感器的具有深度、近红外和激光反射率数据的低分辨率图像完善了数据集。

2、该数据集包括在室内和室外环境中具有基于 MOCAP 的地面实况的序列。据我们所知，这是第一个在森林环境中提供如此准确的地面实况以及室内区域的激光雷达数据集，尽管轨迹长度有限。

3、除了 MOCAP 标记的数据外，该数据集还包括大型室内大厅、道路和森林路径中的其他序列。各种各样的传感器使激光雷达里程计和建图算法之间的比较达到了以前不可能的程度，包括通用方法和传感器特定方法。

4、基于所提供的数据集，我们比较了里程计提升以及使用不同传感器和不同算法获得的地图的质量。
