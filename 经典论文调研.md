# 稀疏视觉

### ORB-SLAM2: an Open-Source SLAM System for Monocular, Stereo and RGB-D Cameras

**摘要：**

​		我们提出了一个完整的SLAM系统，支持单目相机、双目相机、RGB-D相机，拥有地图重用、回环检测、重定位等功能。本系统可以在标准的CPU上实时运行，应用范围从室内小型手持设备的序列，工业环境无人机的序列，再到城市里汽车的序列。我们的后端基于单目视觉和立体视觉的BA，可以使用米制尺度进行精确的轨迹估计。我们的系统包含一个轻量级的定位模式，该定位模式利用视觉里程计跟踪未建图的区域并匹配允许零漂移定位的地图点。我们在29个公共序列上进行评估，结果显示我们的方法达到了SOTA精度，是精度最高的SLAM解决方案。

**论文：**https://arxiv.org/pdf/1610.06475.pdf

**代码：**https://github.com/raulmur/ORB_SLAM2

**创新点：**

1、这是第一个同时支持单目、双目、RGB-D相机的开源的SLAM系统，包含回环检测、重定位和地图重用等功能。

2、ORB-SLAM2中使用 Bundle Adjustment的 RGB-D模式比基于ICP或者光度和深度最小化的方法精度高。

3、通过使用远近双目点（这里指远近点信息判断利用的trick），ORB-SLAM2的双目模式比最新的直接双目法SLAM更准确；

4、一种轻量级的定位方式，可以在无法建图的情况下高效的重用地图。

### ENFT-SFM:Efficient Non-Consecutive Feature Tracking for Robust SFM 

**摘要：**

​		运动结构(SfM)在很大程度上依赖于特征跟踪。在图像序列中，如果不能很好地处理动态物体造成不连续的特征跟踪、遮挡或图像噪声，则相应的SfM会受到影响。对于大规模场景，这个问题变得更严重，通常需要捕获多个序列来覆盖整个场景。本文提出了一种高效的非连续特征跟踪(ENFT)框架，用于匹配分布在不同子序列甚至不同视频中的特征跟踪。该框架包括解决结构不明显、噪声或图像失真较大时的特征丢失问题，以及快速识别和连接位于不同子序列中的相同特征的步骤。此外，我们还提出了一种有效的基于分段的从粗到精的SfM算法来鲁棒地处理大数据集。

**论文：**https://arxiv.org/pdf/1510.08012.pdf

**代码：** https://github.com/ZJUCVG/ENFT-SfM

**创新点：**

1、本文改进了二次匹配，将其描述为一个包含两个几何约束的能量函数的最小化，这不仅提高了匹配精度，而且减少了计算量。

2、对非连续航迹匹配算法进行了重新设计，将特征匹配和匹配矩阵更新结合起来进行。它对初始化不那么敏感，减少了匹配时间。

3、我们提出了一种新的基于分段的由粗到精的SfM方法，该方法能够在有限的存储空间内对大数据进行高效的全局优化

### OpenVSLAM: A Versatile Visual SLAM Framework

**摘要：**本文介绍了一个具有高可用性和可扩展性的视觉SLAM框架OpenVSLAM。可视化SLAM系统对于AR设备、机器人和无人机的自主控制等是必不可少的。然而，传统的开源视觉SLAM框架没有设计为第三方程序所调用的库。为了解决这种情况，我们开发了一个新颖的视觉SLAM框架。这个软件设计得很容易使用和扩展。它结合了几个有用的特性和功能，用于研究和开发。

**论文：**https://arxiv.org/pdf/1910.01122.pdf

**代码：**https://github.com/zm0612/openvslam-comments

**创新点：**

1. 它与各种类型的相机型号兼容，并可针对可选的相机型号进行定制；
2. 可以对特征点地图进行存储和加载，然后可以使用预先创建的地图进行重定位；
3. 为了方便用户，提供了在Web浏览器上运行的跨平台查看器。

**视频demo:** https://www.youtube.com/watch?v=Ro_s3Lbx5ms



### TagSLAM: Robust SLAM with Fiducial Markers

**摘要：**

​		使用AprilTag基准标记SLAM，TagSLAM提供了一种方便、灵活和鲁棒性的方法。通过一些简单的提取，TagSLAM为GTSAM因子图优化器提供前端，使得可以快速设计一系列基于标签的实验：full  SLAM、无重叠视图的相机标定、地面实况视觉定位、闭环检测、姿态估计等。本文中详细讨论了TagSLAM如何鲁棒的初始化因子图，并给出了一个应用实例。

**论文：**http://arxiv.org/pdf/1910.00679v1.pdf

**开源代码：**https://berndpfrommer.github.io/tagslam_web

**创新点：**

1、详细的描述了如何实现鲁棒的初始化，使TagSLAM成为一个无需调整参数就可以应用的工具。

**视频demo:**https://berndpfrommer.github.io/tagslam_web/
